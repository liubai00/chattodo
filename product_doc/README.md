# Product Documentation

这个目录存放 AI Todo 的产品文档，用于说明产品定位、版本目标、迭代范围和研发讨论依据。

## 文件说明

- `AI-Todo-MVP-Kickoff-Brief.md`：MVP 启动说明，定义第一版 demo 的产品方向、验证目标、核心流程和实现边界。
- `AI-Todo-PRD.md`：长期产品需求文档，作为 AI Todo 的产品基线，描述产品定位、边界、核心概念和主要能力。
- `AI-Todo-v0.2-Iteration-Requirements.md`：v0.2 迭代需求文档，描述第一版 demo 之后的阶段目标、内部试用闭环和本轮功能范围。

## 阅读顺序

1. 先读 `AI-Todo-MVP-Kickoff-Brief.md`，理解最小可验证版本。
2. 再读 `AI-Todo-PRD.md`，理解长期产品方向。
3. 最后读 `AI-Todo-v0.2-Iteration-Requirements.md`，理解当前迭代要落地的功能。
